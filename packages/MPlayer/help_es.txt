COMANDOS UTILES
m  : Mostrar/Ocultar el Menu
o  : Mostrar temporizador OSD
s  : Cambiar/Volver de salida de TV
v  : Ocultar/Mostrar Subtitulos
Izquierda : Cancelar / Atras
Derecha : Aceptar / Adelante
Abajo : Atras 6x
Arriba : Adelante 6x
+/-  : Retardo del sonido +/- 100ms
O/9 : Volumen +/- 1

COMMANDS IN FILE SELECTION
d  : set destination directory
c  : copy to destination
r  : ask to remove file
o  : confirm file removal
p  : play the entire directory
