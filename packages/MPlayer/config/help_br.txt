COMANDOS UTEIS
m  : Mostra/Esconde o Menu
o  : Mostra o Tempo de Reproducao
s  : Ativa Saida para TV
v  : Mostra/Esconde Legenda
Esquerda : Cancela / Retorna
Direita : Ok / Avanca
Para Cima : Rebobinar 6x
Para Baixo : Avancar 6x
+/-  : Atraso de Audio +/- 100ms
O/9 : Volume +/- 1

COMANDOS DE SELECAO DE ARQUIVOS
d  : configura diretorio de destino
c  : copiar para o destino
r  : pedir para remover arquivo
o  : confirmar remocao de arquivo
p  : reproduzir todo diretorio
l  : recursively view the entire directory
