Полезни команди
m  : Покажи/Скрий Менюто
o  : Покажи OSD Таймер
s  : Включи ТВ-Изход
v  : Покажи/Скрий Субтитри
Ляво : Откажи / Върни се
Дясно : ОК / Напред
Долу : Назад 6x
Горе : Напред 6x
+/-  : Забави звука +/- 100ms
O/9 : Усилване +/- 1

Команди при избиране на файлове
d  : избери целева директория
c  : копирай в целта
r  : питай да премахнеш файл
o  : потвърди премахване на файл
p  : пусни цялата директория
